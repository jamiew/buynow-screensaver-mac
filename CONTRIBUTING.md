# Contributing to BuyNow Screensaver

Thanks for your interest in contributing to this BeOS tribute project!

## How to Contribute

### Bug Reports
- Check existing issues first
- Include macOS version and steps to reproduce
- Screenshots/videos welcome

### Feature Ideas
- Keep it in the spirit of BeOS simplicity
- Consider if it honors the original screensaver's aesthetic

### Code Contributions
1. Fork the repo
2. Create a feature branch
3. Test on your Mac
4. Submit a PR with clear description

## Development

```bash
# Test screensaver changes
./build.sh && ./install.sh

# Test app changes
./build-app.sh && open "BuyNow Screensaver.app"
```

## Code Style
- Keep it simple and readable
- Comment why, not what
- Match existing style

## Credits
Always credit contributors in commits and release notes.
